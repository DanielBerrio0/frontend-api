# Frontend para API de Gestión de Campeones del Mundo

Este proyecto corresponde al frontend desarrollado para consumir y gestionar los servicios de la API de campeones del mundo. Su propósito es proporcionar una interfaz web que permita a los usuarios interactuar de forma sencilla con los endpoints del backend previamente desplegado.

Descripción

La aplicación permite realizar operaciones relacionadas con los países campeones del mundo de fútbol y el año en que obtuvieron el título. Para un uso adecuado, se recomienda primero registrar un usuario y posteriormente iniciar sesión. Una vez autenticado, el sistema habilita el acceso a las funcionalidades completas de la plataforma.

Entre las acciones disponibles se encuentran:

Registro e inicio de sesión de usuarios

Visualización de los países campeones del mundo

Agregar nuevos registros

Editar registros existentes

Eliminar registros

La API utilizada para este proyecto se encuentra desplegada y funcional. El despliegue se realizó utilizando Railway, permitiendo un acceso estable y continuo para el consumo de datos desde el frontend.

Tecnologías Utilizadas

HTML, CSS y JavaScript

Consumo de API mediante Fetch

Railway para despliegue del backend

Gestión de control de versiones con Git y GitHub, incluyendo el uso de branches para desarrollo y actualización progresiva de funcionalidades

Estado del Proyecto

El frontend y la API se encuentran en funcionamiento. Las funcionalidades principales están implementadas y conectadas correctamente al backend desplegado.

Uso

Registrarse en la plataforma mediante el formulario disponible.

Iniciar sesión con las credenciales creadas.

Una vez autenticado, acceder a la sección de gestión de campeones del mundo.

Realizar las operaciones deseadas: agregar, editar, eliminar o visualizar registros.
